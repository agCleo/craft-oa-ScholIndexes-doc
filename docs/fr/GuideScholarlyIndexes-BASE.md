# BASE

Indexe les métadonnées de toutes sortes de ressources scientifiquement pertinentes - revues, entrepôts institutionnels, collections numériques, etc. - qui disposent d'une interface OAI et utilisent l'OAI-PMH pour fournir leur contenu.

| Element | Description |
| :- | :- |
| Nom | BASE |
| Site web | https://www.base-search.net/ |

| Type de propriétaire | Organisme de recherche public |
| Pays du propriétaire | Allemagne |
| Date de lancement | 2004 |
| Périmètre | Multidisciplinaire |

| Accès pour les founisseurs de données | Gratuit |

## SERVICE FOURNI

| Element | Description |
| :- | :- |
| Type de service | Agrégateur automatisé avec soumission |
| Type de contenu | Contenu scientifique uniquement (vérifié par BASE) |
| Langue du contenu | Tout langage |
| Provenance géographique du contenu | Toute provenance |
| Niveau d'indexation des publications | Article et équivalents |
| Full text | URL vers le texte intégral quand elle est disponible |
| Sources de l'index | Entrepôt OAI-PMH <br>OpenArchives<br>ROAR<br>OpenDOAR<br>Quelques instances DSpace<br>Quelques instances OJS<br>Crossref |
| Données collectées | Métadonnées |
| Standards supportés | Dublin Core |
| Bibliodiversité | Recherche multilingue<br>Aucune limitation apparente dans l'indexation multilingue (restriction aux seuls alphabets latins ?) |
| Services additionnels | API de BASE : http://oai.base-search.net/  |

| Element | Description |
| :- | :- |
