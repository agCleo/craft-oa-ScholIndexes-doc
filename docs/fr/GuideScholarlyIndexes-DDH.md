# DDH

Le Diamond Discovery Hub (DDH) est l'un des services de l'European Diamond Capacity Hub (EDCH). Le Diamond Discovery Hub (DDH) vise à devenir un registre exhaustif des revues en accès ouvert sans frais de publication, publiées en Europe par des institutions publiques ou à but non lucratif et gouvernées par la communauté scientifique. Le DDH a pour objectif d'améliorer la visibilité et la découvrabilité des revues en accès ouvert de type « diamant ».

## PROCESSUS D'INCLUSION

| Element | Description |
| :- | :- |
| Options d'inclusion | Le DDH sélectionne ses sources de confiance selon un processus en deux étapes, décrit ici : https://ddh.diamas.org/docs/become-a-trusted-source/. <br><br>La première étape consiste à s'inscrire au registre EDCH : https://registry.diamas.org/register. <br>Ce registre reconnaît et accepte différents types d'acteurs : éditeurs institutionnels, prestataires de services d'édition, fournisseurs de services, d'outils et de technologies, ainsi que des Centres de Ressources nationaux (National Capacity Centers).<br><br>La seconde étape consiste à valider l'organisation inscrite en tant que source de confiance du DDH. L'équipe du DDH effectue des vérifications supplémentaires concernant l'organisation (la documentation relative aux étapes de vérification est en cours d'élaboration).<br><br>Une fois validée, la source de confiance doit accepter les conditions d'utilisation du DDH et assumer l'entière responsabilité de vérifier la conformité de sa revue aux six critères Diamant du DDH.<br><br>Le DDH ne vérifie pas a posteriori l'exactitude des informations transmises par les sources de confiance, car il s'appuie entièrement sur elles pour ce travail.<br><br>Voici les liens utiles concernant les sources fiables :<br><br>- Conditions d'utilisation : https://ddh.diamas.org/docs/terms-of-service/ <br><br>- Critères Diamond : https://ddh.diamas.org/docs/diamond-journals/ <br><br>- Lignes directrices pour la validation des critères : https://ddh.diamas.org/docs/diamond-open-access-journal-verification-guidelines/ <br><br>- FAQ sur la validation des critères : https://ddh.diamas.org/docs/faq/  |
| Processus de collecte des données  | La collecte des données peut s'effectuer de différentes manières :<br>- importation d'une feuille de calcul Excel (au format JMEF)<br>- remplissage d'un formulaire en ligne (pour chaque revue)<br>- pour les revues sur OJS : importation au format JMEF via un plugin dédié<br>- extraction, via OAI-PMH, d'un ensemble de données au format JMEF <br><br>Les différentes options d'importation des données sont décrites ici : https://ddh.diamas.org/docs/data-ingestion/.  |

## CONDITIONS MINIMALES

| Element | Description |
| :- | :- |
| Critères éditoriaux | Les principaux critères éditoriaux correspondent aux six critères Diamant du DDH :<br>- Identification pérenne : la revue doit posséder un ISSN enregistré.<br>- Revue scientifique : il doit exister un processus d'évaluation par les pairs.<br>- Accès ouvert avec licence ouverte : les publications doivent être en accès ouvert et comporter une licence ouverte.<br>- Gratuité : aucun frais n'est exigé des auteurs ni des lecteurs.<br>- Ouverture à tous les auteurs : aucune affiliation spécifique n'est requise pour publier.<br>- Gouvernance communautaire : la revue doit appartenir à des organismes publics ou à but non lucratif (ou à des compoosantes de ceux-ci) dont la mission comprend la réalisation ou la promotion de la recherche et de l'enseignement. |
| Formats de données | Non applicable |
| Standards de métadonnées | Le format standard utilisé par le DDH est le Journal Metadata Exchange Format (JMEF).<br>Il est décrit en détail sous forme de schéma XML ici : https://github.com/CeON/jmef. <br>Son contenu est également disponible ici : https://ddh.diamas.org/docs/ddh-metadata/.  |
| Formats de métadonnées | Fichier Excel importé directement<br>HTML importé via un formulaire web<br>XML importé via la récupération OAI |
| Métadonnées minimales | Titre<br>ISSN électronique<br>Critère de revue scientifique<br>Critère de gratuité<br>Critère d'accès ouvert<br>Critère d'ouverture à tous les auteurs<br>Critère de gouvernance communautaire |
